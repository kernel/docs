---
title: "Zed"
description: "Connect Zed to the Kernel MCP server"
---

**Using Kernel CLI (recommended):**

```bash
kernel mcp install --target zed
```

**Manual setup:**

1. Press **⌘/Ctrl ,** to open settings.
2. Paste:

```json
{
  "context_servers": {
    "kernel": {
      "source": "custom",
      "command": "npx",
      "args": ["-y", "mcp-remote", "https://mcp.onkernel.com/mcp"]
    }
  }
}
```
